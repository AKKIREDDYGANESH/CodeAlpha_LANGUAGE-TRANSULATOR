# LANGUAGE-TRANSULATOR

Translator App using Tkinter and Google Translate API
This is a simple translator application built using Tkinter and the Google Translate API. The application allows users to input text in a source language, select the source and destination languages, and then translates the input text to the desired language using the Google Translate API.

Installation
1. Cloone: git clone https://github.com/your-username/translator-app.git
2. Navigate to the project directory: cd translator-app
3. Install the required packages using pip: pip install -r requirements.txt

Usage
Run the application:
Copy code: python translator_app.py
1. Enter the text you want to translate in the "Source Text" text box.
2. Select the source language from the drop-down menu.
3. Select the destination language from the drop-down menu.
4. Click the "Translate" button to see the translated text in the "Destination Text" text box.

Features
1. User-friendly graphical interface.
2. Supports translation between various languages.
3. Utilizes the Google Translate API for accurate translations.
